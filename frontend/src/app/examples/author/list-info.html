General info

This page demonstrates fetching data from the backend via WebSockets and showing it in a simple list. This example application fetches authors with related book data populated for each and also covers data pagination, sort, and search functions. The actual data is fetched from the following endpoint on the backend. 

GET {{backendConfig.url}}/author

Note that all data communication to the backend requires a JSON Web Token (JWT) for authentication to make sure that user is allowed to access the data.

Functions in this example

Data fetch
From backend via WebSockets using $sailsSocket service.
Search
Specify which columns are used in search and type your search words to see results while you're typing.
Pagination
By default, the page displays an example of ten books per page. You can change the number to be displayed per page.
Sorting data
Simply by clicking the column header. This will fetch data from backend again. Unfortunately not from relation columns... yet.
Live updates
GUI is automatically updated if changes are made to the data.
Navigation
Access to a single author and book pages simply by clicking the author or book name.
